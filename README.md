# 3dgp-ex-dark-pictures

# Genre
Survival horror

# Beschrijving (nog te updaten, specifieren)
Je bent een fotograaf die geruchten heeft gehoord over een oud gebouw. Je moet fotos maken van interessante dingen.
Je moet je navigeren door donkere kamers terwijl er een entity je achterna loopt. Het is zo donker dat je bijna niets ziet. Enkel door de flash van je fototoestel zie je waar je loopt.
Vind de interessante dingen, maak er fotos van en ontsnap van de entity.

# Setting
Je bevind je in een donker verlaten gebouw (nog te specifieren waar), je voelt rillingen door je rug. Je hoort rare geluiden rondom jou.

# unique selling point
De environmnet is zo donker waardoor je enkel door het gebruiken van de korte flash je weg vindt. Dit maakt de ervaring nog enger en spannender.


# Gameplay voorbeeld
Je komt in een gebouw met een lijst van dingen die je moet fotograferen. In de donkere kamers is niets te zien. Je gebruikt je flashlight om kort iets te zien. Opeens zie je rode ogen, een groot monster dat je achterna loopt. Je zoekt snel de dingen die je moet fotograferen. Probeer levend te onstappen uit het gebouw met de getrokken fotos.

Inspiratie van FNAF, outlast.
